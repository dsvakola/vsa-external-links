WebOrbit — Changelog

Version 1.11 — 2025-11-24
* Added: Whitelist wildcard support for IPs and domains (supports * and ? patterns)
* Added: Per-role access control UI and backend support (define rules by WP role)
* Improved: Rule matching performance and caching
* Fixed: Edge-case where empty-rule sets blocked admin dashboard for some installs
* Fixed: UI layout issues on small screens
* Docs: Updated README.md and readme.txt with new features and upgrade steps

Version 1.10 — 2025-08-07
* Added: Global whitelist and basic logging
* Fixed: Compatibility with caching plugins (improved headers)
* Improved: Fallback behavior for unknown requests

Version 1.0 — 2025-04-14
* Initial public release: core request filtering and whitelist functionality
